☕ Coffee Shop Sales Analysis using Python

📌 Project Overview
This project analyzes a Coffee Shop Sales dataset using Python, Pandas, Matplotlib, and Seaborn to uncover sales trends, customer purchasing behavior, store performance, and product insights. The analysis combines data preprocessing, exploratory data analysis (EDA), pivot tables, and multiple visualizations to support business decision-making.
________________________________________
🎯 Objectives
•	Analyze sales performance across different store locations.
•	Identify the highest-selling product categories and product types.
•	Compare monthly and weekday sales trends.
•	Visualize customer purchasing patterns using different charts.
•	Generate insights that can help improve sales .
________________________________________
🛠️ Technologies Used
•	Python
•	Pandas – Data Cleaning & Analysis
•	Matplotlib – Data Visualization
•	Seaborn – Statistical Visualizations

________________________________________
📊 Key Analysis Performed
•	✔️ Created a Month column from transaction dates.
•	✔️ Analyzed sales by Store Location.
•	✔️ Compared sales across Months and Weekdays.
•	✔️ Identified Top 5 Product Categories by revenue.
•	✔️ Identified Top 10 Product Types by quantity sold.
•	✔️ Built Pivot Tables for store-wise and weekday-wise sales.
•	✔️ Used user input to filter data by store location and product category.
•	✔️ Generated descriptive statistics and correlation analysis.
________________________________________

📈 Visualizations Included
 Count Plot , Bar Plot , Line Plot, pie Chart, Histogram,  Point Plot,  Strip Plot,  Violin Plot, Swarm Plot, Scatter Plot, Pair Plot,  Joint Plot and KDE Plot. These visualizations help understand sales distribution, product demand, pricing patterns, and relationships between different variables.
________________________________________
💡 Key Insights
•	Multiple store locations contribute differently to total revenue.
•	A few product categories generate the majority of sales.
•	Customer purchasing patterns vary across weekdays and months.
•	Product pricing influences total sales and transaction quantities.
•	Visual analysis helps identify business trends, popular products, and sales opportunities.
